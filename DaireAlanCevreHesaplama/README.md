## Dairenin Alanını ve Çevresini Hesaplayan Program

Java ile yarı çapını kullanıcıdan aldığınız dairenin alanını ve çevresini hesaplayan programı yazın.
- Not `𝜋` sayısını `3.14` alınız.

# Formül

- `Alan Formülü` = π * r * r;
- `Çevre Formülü` = 2 * π * r;
- `Alan * Alan` = 𝑢 * (𝑢 − 𝑎)* (𝑢 − 𝑏) * (𝑢 − 𝑐)
